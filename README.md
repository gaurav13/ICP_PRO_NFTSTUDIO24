# Nftstudio24 decentralized media platform.

### Install dfx

```sh
sh -ci "$(curl -fsSL [https://internetcomputer.org/install.sh](https://internetcomputer.org/install.sh))"
```

### Install Packages

```bash
yarn install
```

### Running Nftstudio24

```bash
dfx deploy

dfx start 

yarn run dev
```
